Why ACEM Project‿Sphere?

Transforming the way ACEM handles project submissions ✨

🎓 For Students

  • 📤 Upload all project files (code, PPTs, reports, videos)
  • 💬 Chat directly with mentors
  • 📄 View submission history

👩‍🏫 For Mentors

  • 🔍 View and review assigned student submissions
  • ✅ Mark projects as complete
  • 💭 Provide feedback directly

🛠️ For Admin

  • 👥 Manage all users (add/remove/update roles)
  • 🔗 Assign and reassign mentors to students
  • 📊 View analytics on project progress

Centralized Uploads
